- TypeORM 0.3.x 에서 연관된 Entity 에 대한 조회 조건을 지정할 때

```jsx
// Given: board.user <-> user.boards

async getAllBoards(user: User): Promise<Board[]> {
    return await this.repository.find({
      where: {
        user: {
          id: user.id,
        },
      },
    });
  }
```
